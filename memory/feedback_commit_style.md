Commit mesajlarının sonuna "Co-Authored-By: Claude" veya "AI ile yapılmıştır" gibi ifadeler ekleme. Commit author olarak sadece kullanıcı görünmeli.

**Why:** Kullanıcı kendi projesinde sadece kendisinin görünmesini istiyor, AI imzası istemedi.

**How to apply:** git commit yaparken standart mesaj kullan, hiçbir AI imzası veya ek satır ekleme.
